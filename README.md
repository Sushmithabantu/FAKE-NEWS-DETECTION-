# Fake News Detection using NLP

## Overview
This project builds a machine learning model to classify news articles as Fake or Real using NLP techniques.

## Techniques Used
- TF-IDF Vectorization
- Logistic Regression
- Naive Bayes

## Results
Achieved high accuracy in classification and analyzed key features influencing predictions.

## Dataset
Download from:
https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset

## How to Run
1. Download dataset
2. Place CSV files in same folder
3. Run the notebook
